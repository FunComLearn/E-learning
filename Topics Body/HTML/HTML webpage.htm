Hypertext markup language OVERVIEW

 

What is:

 

 

Hypertext Markup Language is the language used on websites to display fonts, colors, links, and images to site visitors. Within the speech, several tags enable the site creator to customize the way the site looks. These tags and codes are then read by Internet browsers, which display the website according to the specifications listed in the HTML codes. Visitors to the site then view the sites pages as the creator designed them without seeing the list of regulations and tags.
